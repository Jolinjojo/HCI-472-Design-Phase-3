Typography

Basic & Roboto

This typographical hierarchy shows one serif (Tiro Kannada) and one sans-serif (Fenix) typefaces. Tiro Kannada provides an elegant and simple look to headings, while Fenix compliments it, serving as legible body typeface. This typographical hierarchy is arranged using a Perfect Fourth Scale with a base font size of 25px.
